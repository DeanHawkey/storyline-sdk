Packagecom.articulate.wg.v2_0
Classpublic class wgEventFrame
InheritancewgEventFrame flash.events.Event

Represents events that are specific to a frame that implements the wgIFrame interface. In addition, a wgEventFrame event can be used by the frame to obtain objects from the player. In this case, after the event has been dispatched, the requested object can be accessed through the ReturnData property of the event.

ReturnDataproperty 
ReturnData:Object

The ReturnData is set by the player when it respondes to the dispatched event. It will vary, depending on the type of event dispatched.

FRAME_READY = null

GET_PLAYER = a reference to the IPlayer object

GET_SLIDEDRAW = a reference the ISlideDraw object with the specified Id

GET_SLIDE = a reference to the ISlide object with the specified Id

GET_SLIDEBANK_SLIDE = a reference to the ISlide object with the specified Id or null if the slide has not been created

GET_CURRENT_SLIDE = a reference to the ISlide object for the current slide

GOTO_SLIDE = null

TRIGGER_CUSTOM_EVENT = null
